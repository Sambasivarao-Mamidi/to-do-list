# TaskMaster - To-Do List

A modern, interactive to-do list application built with HTML, CSS, and JavaScript. Perfect for learning DOM manipulation while building a practical application.

## Features

- **Add Tasks** - Enter task text and press Add or hit Enter
- **Mark as Done** - Click anywhere on a task to mark it complete
- **Delete Tasks** - Click the Delete button to remove a task
- **Priority Levels** - Choose High, Medium, or Low priority for each task
- **Filter Tasks** - View All, Pending, or Completed tasks
- **Statistics** - See total, completed, and pending task counts
- **LocalStorage** - Tasks persist even after refreshing the page
- **Clear Completed** - Remove all completed tasks at once
- **Responsive Design** - Works on desktop and mobile devices

## Quick Start

1. Open `index.html` in your web browser
2. Start adding tasks!

No installation or server required.

## Learning Objectives

This project demonstrates core DOM manipulation concepts:

- `document.createElement()` - Create new elements dynamically
- `appendChild()` - Add elements to the DOM
- `addEventListener()` - Handle user interactions
- `classList.toggle()` - Toggle CSS classes
- `localStorage` - Save data in the browser
